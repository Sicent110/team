#### tbCompany 表 
| 字段名 | 类型 | 说明 | 
|:--------:|:---------:|:-------:| 
| id | int(11) unsigned | 主键 | 
| fdName | varchar(32) | 公司名称 | 
| fdDescription | varchar(255) | 描述 | 
| fdStatus | smallint(1) | 1可用，2已删除 | 
| fdCreate | datetime | 创建时间 | 
| fdUpdate | datetime | 更新时间 |
